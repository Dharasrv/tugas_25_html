Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, article bisa kita gunakan element header dan footer.
    Karena tag article memilikii type block, artinya elent ini memiliki lebar sesuai lebar browser jika tidak menentukan lebarnya.

  • Aside

    • Element ini biisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bahwa posting untuk mengjelaskan identitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai pagejika kita tidak menentukan lebarnya.
